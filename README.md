# 🎰 Slot Machine Game (Python)

A simple command-line based Slot Machine game built using Python. This project simulates a real slot machine where users can deposit money, place bets on multiple lines, and spin to try their luck.

---

## 🚀 Features

- Deposit money to start playing  
- Bet on 1 to 3 lines  
- Set bet amount per line  
- Random slot generation using Python  
- Winning system based on matching symbols  
- Displays winning lines and total winnings  
- Continuous gameplay with balance tracking  

---

## 🎮 How the Game Works

- The slot machine has a **3x3 grid** (3 rows × 3 columns)  
- Each spin randomly generates symbols for each column  
- The player selects the number of lines and betting amount  
- If all symbols in a selected row match, the player wins  
- Winnings are calculated based on symbol value and bet  

---

## 🎲 Symbol Details

| Symbol | Count | Value |
|--------|------|-------|
| A      | 2    | 5x    |
| B      | 4    | 4x    |
| C      | 6    | 3x    |
| D      | 8    | 2x    |

- Higher-value symbols are rarer, making the game balanced and interesting.
